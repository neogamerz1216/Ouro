# Ouro Browser

**Ouro** is a fast, minimal, and privacy-focused web browser built for a cleaner internet experience. Developed by **Pavan** from Team Ouro.



##  Features
* **Privacy First:** Built-in ad and tracker blocking.
* **Minimalist Design:** Focus on your content, not the browser UI.
* **Fast & Lightweight:** Optimized for speed and low resource usage.
* **Open Source:** Built on the solid foundation of the Min browser engine.

##  Installation
1. Go to the [Releases] (https://github.com/neogamerz1216/Ouro/releases) page.
2. Download the latest `ouro-setup.exe`.
3. Run the installer and start browsing!

*Note: Since this is an independent build, Windows may show a "SmartScreen" warning. Click **More Info** -> **Run Anyway** to proceed.*

## 🛠️ Development

If you want to build Ouro from source:

*DEVELOPMENT*

# Clone the repo
git clone [https://github.com/neogamerz1216/Ouro.git](https://github.com/neogamerz1216/Ouro.git)

# Install dependencies
npm install

# Build for Windows (requires Docker & Wine)
npm run buildWindows
